# Nearby Essentials Finder

A web application that helps users quickly find **nearby essential services** such as hospitals, pharmacies, grocery stores, ATMs, and more based on their location.

This project is built using the **MERN stack** and focuses on solving real-world accessibility problems by providing location-based service discovery.
